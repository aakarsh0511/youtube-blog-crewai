# YouTube Blog Generator using CrewAI

## Overview

This project is a Multi-Agent AI application built using CrewAI that automatically converts YouTube video content into well-structured blog posts. The system uses specialized AI agents to research video content, extract key insights, and generate engaging blog articles.

## Features

* YouTube video content analysis
* Multi-agent collaboration using CrewAI
* Automated blog generation
* AI-powered research and content writing
* Structured and readable blog output

## Tech Stack

* Python
* CrewAI
* YouTube Data/Transcript API
* Large Language Models (LLMs)

## Installation

```bash
pip install -r requirements.txt
```

## Project Workflow

1. Research Agent analyzes YouTube video content.
2. Writer Agent extracts key insights.
3. AI generates a comprehensive blog post based on the video.
